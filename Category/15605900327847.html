数据类型

内存

  • 内存是线性的，按字节为单位进行编址的

补码

  • 0的补码是0
  • 符号位和普通位一并参与运算
  • 补码可以使加减乘除，全部变加法
  • 127 -128

数据类型

  • 数据类型提供了申请内存单元的大小和访问规则，数据类型是假设在线性内存上的一种逻辑关系
  • unsigned扩充前面应该补0
  • -1转换成补码存在计算机中
  • 计算机只识别 0 1，计算机要表达的东西是认为追加的

面试题

int main() {
  char a[1000];
  int i;
  for (i = 0; i < 1000; i++)
     a[i] = -1 - i;
  printf("%d \n", strlen(a));
}
