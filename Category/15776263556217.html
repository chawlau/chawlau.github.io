协程和异步

2019/12/29 21:32 下午 posted in  Python

同步

  • 指代码调用IO操作时，必须等待IO操作完成后才返回的调用方式
  • 异步是指代码调用IO操作时，不必等IO操作完成就返回的调用方式

阻塞和非阻塞

  • 阻塞是指调用函数时候当前线程被挂起。
  • 阻塞是指调用函数时候当前线程不会被挂起，而是立即返回

回调函数

  • 如果回调函数执行不正常该如何？
  • 如如果回调里面还要嵌套回调怎么办？
  • 要嵌套很多层怎么か？
  • 如果嵌套了多层，其中某个环节出错了会造成什么后果？
  • 如果有个数据需要被每个回调都处理怎么办？
  • 怎么使用当前函数中的局部变量？

需求

  • 采用同步的方式去编写异步的代码
  • 使用单线程去切换任务：
  • 线程是由操作系统切换的，单线程切换意味着我们需要程序员自己去调度任务
  • 不在需要锁，并发性高，如果单线程内切换函数，性能远高于线程切换，并发性更高

生成器

  • next 和 send
  • send可以传递值进入生成器内部,同时还可以重启生成器进入到下一个yield位置
def gen_func():
    html = yield "http://projectedu.com"
    print(html)
    yield 2
    yield 3
    return 'bobby'

if __name__ == '__main__':
    gen = gen_func()
    url = next(gen)
    html = "bobby"
    gen.send(html)
  • 第一次调用send的时候不能调用一个非None值，因为生成器还没有执行到那一行代码
  • next则可以
def gen_func():
    html = yield "http://projectedu.com"
    print(html)
    yield 2
    yield 3
    return 'bobby'

if __name__ == '__main__':
    gen = gen_func()
    # url = next(gen)
    gen.send(None)
    html = "bobby"
    print(gen.send(html))

def gen_func():

    except Exception as e:
        pass

if __name__ == '__main__':
