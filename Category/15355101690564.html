网络名称空间

  • ip netns exec r1 ifconfig -a 查看网络名称空间
  • ip netns add r1 添加网络名称空间
  • ip link add name veth1.1 type veth peer name veth1.2
  • ip link set dev veth1.2 netns r1 移动到网络名称空间
  • ip netns exec r1 ip link set dev vrth1.2 name veth1.2 改名
  • ifconfig veth1.1 10.1.0.1/24 up 激活
  • ip netns exec r2 ping 10.1.0.2
